## Technology 

Here are the technologies used in this project.

* JavaScript
* NodeJs
* Express
* Nodemon
* Docker
* MySql
* Sequelize
* Joi
* JWT
* Dotenv

## Services Used

## Features

The main features of the application are:
 - Create an API of a blog
 - Validate user Login
 - List and create user
 - List blog posts
 - List and create post categories
